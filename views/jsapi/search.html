api
初始化插件： search(this,options,emes)
     其中 this：是指要显示这个表格的js选择器(非jQ), 或者$("#f_date")[0](jQ)
          options：参照$.ajax(options),其中应用插件(input)的name会覆盖options.data.name;
          emes :第二个以外要使用数据的input，可以是影藏元素。

    //ajax请求的数据格式
 {
     "status": 0,
     "data": [
         ["aaaaaa",1],
         ["bbbbbbbb",2],
         ["cccccc",3]
     ]
 }
